<objective>
Unblock the Telegram → AgentOS → memory path so MEM-01 (5-second SLA) is verifiable on real traffic, not just synthetic POSTs. Two surgical edits: turn off the broken `reasoning=True` flag on the supervisor team, and raise the telegram adapter's httpx timeout to accommodate slow team runs.

Per `evidence/experiment-2026-05-22.md`: the supervisor team hangs because `reasoning=True` + a non-native reasoning model (`LiteLLMChat` proxying NIM DeepSeek V4 Pro with `thinking:true`) puts Agno into a manual Chain-of-Thought loop that can't parse the thinking-model's output. The httpx client's 35s timeout then ends every real Telegram message as a "Network error". Disabling the team's manual CoT (orchestrator model still produces thinking internally) + raising the budget to 90s fixes both halves.
</objective>
