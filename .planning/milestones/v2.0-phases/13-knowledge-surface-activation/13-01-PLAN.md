<objective>
Rewrite `agentos/knowledge.py` so that:

1. `make_knowledge()` constructs a Knowledge instance with the wiring AgentOS actually requires (`name` + `contents_db`). This is the application-level fix for RC-knowledge-not-registered — once shipped, `/knowledge/config` Available IDs becomes non-empty even before any content is indexed.
2. A pure `reindex(...)` function is the sole writer to the knowledge surface. It walks the vault, computes sha256 of file bytes, skips files whose recorded sha256 matches, and calls `Knowledge.insert(...)` per changed file. Per-file errors are isolated.
3. The CLI entry `python -m agentos.knowledge --reindex` calls `reindex()` and emits human-readable progress + a final summary.
4. Stub fallback (missing DSN, init failure) emits a loud WARNING log line instead of silently returning an empty Knowledge stub.
5. Every write event (indexed, skipped, error) emits an OBS-01 structured log line on the `agentos.knowledge` logger.

This plan is shippable independently. It satisfies KNOW-01 (rows land in ai.vault + ai.agno_knowledge), KNOW-03 (re-running reindex yields identical row count), DIAG-BL-05 (tables exist after first reindex), DIAG-BL-06 (no more silent stub), and the write half of OBS-01.

Plan 13-02 adds the search-path wrapper (InstrumentedKnowledge). Plan 13-03 is the deploy + UI verification closeout.
</objective>

    Rewrite `agentos/knowledge.py` end-to-end. Contract surface required by tests:

    ```python
    from dataclasses import dataclass

    @dataclass
    class ReindexSummary:
        indexed: int
        skipped: int
        errors: int
        total: int
        duration_seconds: float

    def make_knowledge() -> Knowledge:
        """Returns a wired Knowledge instance (or a stub on failure).

        On success: Knowledge(name="ultra-brain-vault", vector_db=PgVector(...),
                              contents_db=POSTGRES_DB).
        On stub fallback: Knowledge(name="ultra-brain-vault") with no vector_db,
                          no contents_db; emits WARNING log line.
        """

    def reindex(vault_path: Path | None = None,
                knowledge: Knowledge | None = None) -> ReindexSummary:
        """Walk vault_path/**/*.md, sha256 each file, skip unchanged, insert changed."""

    def cli_main(argv: list[str] | None = None) -> int:
        """Parses --reindex flag, calls reindex(), prints per-file + summary, returns exit code."""

    if __name__ == "__main__":
        import sys
        sys.exit(cli_main(sys.argv[1:]))
    ```

    Implementation requirements:

    **Logger setup (top of file, mirroring `instrumented_memory.py:13-22`):**
    ```python
    log = logging.getLogger("agentos.knowledge")
    log.setLevel(logging.INFO)
    if not log.handlers and not logging.getLogger().handlers:
        _h = logging.StreamHandler()
        _h.setFormatter(logging.Formatter("%(asctime)s %(levelname)s %(name)s %(message)s"))
        log.addHandler(_h)
        log.propagate = False
    ```

    **`make_knowledge()`:**
    - Read `POSTGRES_DSN_KNOWLEDGE` env. If empty/missing → emit WARNING log line (see schema in test 2) and return `Knowledge(name="ultra-brain-vault")`.
    - Else try to build `PgVector(table_name="vault", db_url=DSN, embedder=SentenceTransformerEmbedder(id="sentence-transformers/all-MiniLM-L6-v2"), search_type=SearchType.hybrid, reranker=SentenceTransformerReranker())`. On exception → WARNING log line with `"reason": str(exc)`, return stub.
    - On success: import `POSTGRES_DB` from `agentos.db` and return `Knowledge(name="ultra-brain-vault", vector_db=vector_db, contents_db=POSTGRES_DB)`.

    **`reindex(vault_path, knowledge)`:**
    - Defaults: `vault_path = vault_path or Path(os.getenv("VAULT_PATH", "vault"))`; `knowledge = knowledge or make_knowledge()`.
    - If `knowledge.vector_db is None` (stub fallback path): log one ERROR `{"path":"knowledge","status":"reindex-unavailable","reason":"knowledge stub — POSTGRES_DSN_KNOWLEDGE not set"}` and return `ReindexSummary(0, 0, 0, 0, 0.0)`.
    - Build `existing_rows` dict via `knowledge.contents_db.get_knowledge_contents()` → `{row.name: row.metadata or {}}`. If `contents_db` lookup raises, log ERROR and treat as empty (graceful — first-ever reindex will hit this when table doesn't exist yet; subsequent runs will work).
    - For each `file_path` in `sorted(vault_path.rglob("*.md"))`:
      - Compute `rel_path = str(file_path.relative_to(vault_path))`.
      - Read bytes; compute `sha256_hex = hashlib.sha256(content_bytes).hexdigest()`.
      - If `existing_rows.get(rel_path, {}).get("file_sha256") == sha256_hex` → log `action="skipped"` OBS-01 line, print `[skipped] {rel_path}`, increment `skipped`.
      - Else:
        - `started = time.monotonic()`
        - try `knowledge.insert(path=str(file_path), name=rel_path, metadata={"file_sha256": sha256_hex, "rel_path": rel_path, "size": len(content_bytes), "indexed_at_ms": int(time.time() * 1000)}, upsert=True, skip_if_exists=False)`
        - On success: increment `indexed`; emit OBS-01 `action="indexed"`; print `[indexed] {rel_path}`.
        - On exception: increment `errors`; emit OBS-01 `action="error"`, `status="error"`, `error_type`, `error_msg` (truncated 200ch); print `[error] {rel_path}: {error_msg}`. Continue loop.
      - `latency_ms = int((time.monotonic() - started) * 1000)` (for indexed/error; for skipped, latency_ms = 0).
      - `db_id = getattr(knowledge.contents_db, "id", None) if knowledge.contents_db else None`.
      - `row_id` = `existing_rows.get(rel_path, {}).get("id")` if skipping, else None for new inserts (we don't have the row id without a second query — acceptable per CONTEXT D-05).
    - Return `ReindexSummary(indexed, skipped, errors, total=indexed+skipped+errors, duration_seconds=time.monotonic() - start_total)`.

    **`cli_main(argv)`:**
    - If `--reindex` not in argv: print usage `"Usage: python -m agentos.knowledge --reindex"`, return 1.
    - Else: `summary = reindex()`. Print final line via `f"Indexed {summary.indexed} files ({summary.skipped} skipped, {summary.errors} errors) in {summary.duration_seconds:.2f}s"`. Return 0 if `errors == 0` else 1.
    - The per-file `[indexed]/[skipped]/[error]` lines are printed inside `reindex()` itself, NOT in cli_main (so tests using `reindex()` directly still see them via capsys — but tests that don't want stdout chatter can monkeypatch `print`).

    **Imports:** keep existing `PgVector`, `SearchType`, `SentenceTransformerEmbedder`, `SentenceTransformerReranker`, `Knowledge`. Add `hashlib`, `time`, `logging`, `json`, `dataclasses.dataclass`. Import `POSTGRES_DB` from `agentos.db`.
