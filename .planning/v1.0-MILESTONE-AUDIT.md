## Tech Debt Summary

**Phase 01 — ultra-brain-agno (5 items):**

1. **Port fallback mismatch** — `telegram_adapter.py:60` default is `7001`, brain binds on `7000`. Systemd env var saves production but breaks any non-systemd dev invocation silently.
2. **Timer race window** — `uab-digest/review/monitor` use `Requires=uab-brain.service` but not `After=`. If brain starts slowly, curl fires before HTTP port is ready.
3. **Route shape drift** — `REQUIREMENTS.md` documents `/v1/agents/{name}` but implementation uses Agno-native `/agents/{agent_id}/runs`. Update REQUIREMENTS.md to reflect actual routes.
4. **REQ-104 binding** — 127.0.0.1 enforced by uvicorn args in `__main__.py` (correct), but `app.py` doesn't set `host` directly. Agno AgentOS class binding behavior unauditable.
5. **Duplicate "2.py" files** — `agentos/app 2.py`, `agentos/db 2.py`, and others are leftover copies. Should be deleted.

## Gaps Analysis

### Critical Blockers (forces `gaps_found` status)

**REQ-204 — Bot Token Not Rotated** *(Explicit Acceptance Criterion)*
- The old bot token has been in use throughout Phase 1 execution, exposed to LLM calls, logs, etc.
- BotFather `/revoke` was listed as the final step before ship in REQUIREMENTS.md acceptance criteria.
- Simple fix: run `/revoke` in BotFather, update `TELEGRAM_BOT_TOKEN` in VPS `.env`, restart `uab-telegram`.

**REQ-011 — Cost Ledger Not Wired**
- `CostLedger` class fully implemented in `ultra_brain/cost.py` — just needs to be called.
- Agno doesn't have a native cost-tracking hook; would require wrapping the LiteLLM response or adding a middleware.
- LiteLLM itself can log costs if `LITELLM_LOG_COST=true` — this might be the path of least resistance.
- Without this, `_system/cost-ledger.md` stays empty forever.

**REQ-200 — deploy/hermes/ Not Deleted**
- `deploy/hermes/config.yaml` still exists locally. Quick fix: `git rm -r deploy/hermes/` + deploy.

**REQ-202 — ultra_brain/llm.py Not Deleted**
- File still present. Agno migration is end-to-end verified (VALIDATION.md). Safe to delete.
- Quick fix: `git rm ultra_brain/llm.py` + commit.

**REQ-203 — Plan File Not Marked Superseded**
- `plans/swift-sniffing-meerkat.md` needs a status header added: `> **Status: SUPERSEDED** — replaced by Phase 1 (ultra-brain-agno) using Agno.`
- Two-minute fix.
