def monthly_telos_recheck(
    vault_root: Path,
    *,
    drift_threshold: float = 0.5,
    send_telegram: bool = True,
    chat_id: str | None = None,
) -> list[dict]:
    """Score all vault/00-Projects/ directories against telos goals.

    Returns list of dicts: {"project": str, "score": float, "rationale": str, "drifting": bool}
    Sends Telegram message listing drifting projects if send_telegram=True and drift found.
    """

Logic:
1. Iterate vault_root / "00-Projects" with iterdir(); collect dirs only.
2. For each project dir, call score_alignment(project.name, vault_root / "_system") to get TelosCheck.
3. Mark drifting = check.score < drift_threshold.
4. Sort results: drifting projects first, then by score ascending.
5. Build a summary report string for stdout:
   - Header: "Monthly TELOS Recheck — {date.today().isoformat()}"
   - Per project: "{project.name}: {score:.2f} {'[DRIFT]' if drifting else '[ok]'} — {rationale}"
   - Footer: "Drifting: {N} / {total}"
6. If any drifting projects AND send_telegram:
   - Build Telegram message: "⚠️ Monthly TELOS Drift Report\n\n{drift_lines}\n\n{N} project(s) drifting from quarter goals."
   - Send via the Telegram send function found in telegram.py. Use send_message_with_buttons with an empty buttons list [] if no dedicated send_telegram_message exists.
7. Return the results list.

Also print the summary report to stdout always (regardless of send_telegram flag).

Add module docstring: "Monthly TELOS recheck — scores vault/00-Projects/ against quarter-goals and flags drift."
