<objective>
Make all enrolled repos queryable from Telegram/anywhere by compiling nightly LLM prose summaries via GitHub Actions and landing them in the second-brain vault as `repos/<name>.md` files.

The codebase-memory-mcp graph is Mac-local — it cannot serve the VPS Telegram bot or teammates. This phase replaces the current useless `reindex_bridge.sh` vault-write (raw stats JSON) with a structured LLM prose summary per repo, compiled in CI.

**Split strategy:** Local file changes (tasks 1–3) are fully automated. External repo setup (checkpoints A–C) requires human action in GitHub UI or a separate repo context. Smoke verification (checkpoint D) closes the loop.
</objective>

<task type="checkpoint:human-action" gate="blocking">
  <name>Checkpoint B: Add aggregate.yml to second-brain repo</name>
  <why-manual>Requires committing to caiobellizzi/second-brain — outside this executor's write access.</why-manual>
  <steps>
    Create .github/workflows/aggregate.yml in caiobellizzi/second-brain:

    name: Aggregate Brain Summaries
    on:
      schedule: [{cron: "0 4 * * *"}]
      workflow_dispatch:
    jobs:
      aggregate:
        runs-on: ubuntu-latest
        permissions:
          contents: write
        steps:
          - uses: actions/checkout@v4
          - name: Download summaries
            uses: dawidd6/action-download-artifact@v6
            with:
              github_token: ${{ secrets.GITHUB_TOKEN }}
              workflow: summary.yml
              repo: caiobellizzi/ultra-agents-brain
              name_is_regexp: true
              name: summary-.*
              path: /tmp/summaries/
          - name: Write repos/*.md
            run: |
              mkdir -p repos
              for dir in /tmp/summaries/*/; do
                repo=$(basename "$dir" | sed 's/^summary-//')
                cp "$dir/SUMMARY.md" "repos/${repo}.md"
              done
          - name: Commit and push
            run: |
              git config user.name "brain-bot"
              git config user.email "noreply@github.com"
              git add repos/
              git diff --cached --quiet || git commit -m "chore: nightly brain summary update"
              git push

    Note: dawidd6/action-download-artifact@v6 can download cross-repo artifacts with a PAT if GITHUB_TOKEN lacks cross-repo access. Add a PAT as GH_PAT secret if needed.
  </steps>
  <resume-signal>Type "aggregate ready" when aggregate.yml is committed to second-brain.</resume-signal>
</task>
