# Wave 5 Plan 1 — Eval Scaffolding (Commit 5a)

## Objective

Create the `evals/` module tree, shared fixtures, 6 dataset stub files, and one smoke test per agent (pytest assertion only, no LLM judge). Goal: `pytest evals/ -k smoke` passes green at zero API cost.

## Context

Agno 2.6.7 provides 4 eval classes (`AccuracyEval`, `ReliabilityEval`, `PerformanceEval`, `AgentAsJudgeEval`) in `agno.eval`. No CLI runner, no built-in dataset format. We build the orchestration as plain pytest.

This commit establishes the scaffold. Commit 5b (plan 07-02) fills in the real test coverage. Commit 5c (plan 07-03) freezes baselines and wires pre-commit routing.

**Judge model default:** `private-worker` = LM Studio (free, offline). Override per-run: `EVAL_JUDGE_TIER=orchestrator pytest evals/` for cloud-tier judges.

### 2. Create `evals/conftest.py`

```python
import os
import pytest
from agentos.model import chat_model
from agno.storage.postgres import PostgresDb

EVAL_JUDGE_TIER = os.getenv("EVAL_JUDGE_TIER", "private-worker")
POSTGRES_DSN_SESSIONS = os.getenv("POSTGRES_DSN_SESSIONS")


@pytest.fixture(scope="session")
def judge_model():
    return chat_model(EVAL_JUDGE_TIER)


@pytest.fixture(scope="session")
def eval_db():
    return PostgresDb(db_url=POSTGRES_DSN_SESSIONS, create_schema=True)
```

### 3. Create dataset stub files

Each file in `evals/datasets/` returns a list of 3 test cases. For this commit, cases are stubs with `TODO` markers — real cases are authored in commit 5b.


Create similarly for: `curator_cases.py`, `ingest_cases.py`, `query_cases.py`, `research_cases.py`, `supervisor_routing.py`.

### 4. Create one smoke test per agent

Each test file gets one `@pytest.mark.smoke` test that instantiates the agent and verifies it has the required kwargs set (no actual run, no LLM call):

```python
# evals/test_chat.py
import pytest
from agentos.schemas import ChatReply

@pytest.mark.smoke
def test_chat_agent_has_output_schema(chat_agent):
    assert chat_agent.output_schema == ChatReply

@pytest.mark.smoke
def test_chat_agent_has_search_knowledge(chat_agent):
    assert chat_agent.search_knowledge is True
```

Repeat for each agent file.

### 5. Wire into Makefile

Add to `Makefile` (or create one):
```makefile
test:
    rtk pytest tests/ evals/

eval-smoke:
    rtk pytest evals/ -k smoke

eval-full:
    EVAL_JUDGE_TIER=orchestrator rtk pytest evals/
```
