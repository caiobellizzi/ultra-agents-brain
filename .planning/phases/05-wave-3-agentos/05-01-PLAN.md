# Wave 3 — AgentOS Surface: PostgresDb, MemoryManager, MCP, A2A

## Objective

Rewire `agentos/app.py` to use `PostgresDb` (replacing SQLite), instantiate the shared `MemoryManager`, pass `memory_manager` and `knowledge` into all agent factories, and expose MCP and A2A protocol endpoints.

This is the central wiring wave. After this wave, all agents use Postgres for session/memory storage, and the AgentOS is discoverable via MCP and A2A.

## Context

**Critical gotcha:** `enable_mcp_server=True` must appear on BOTH `AgentOS(...)` AND `get_app(enable_mcp_server=True)` — one without the other silently disables MCP. This was confirmed in the docs-researcher pass.

MCP endpoint: `http://localhost:7000/mcp` (auth via `OS_SECURITY_KEY` env var in dev).
A2A endpoint: `http://localhost:7000/a2a/agents/{id}/.well-known/agent-card.json`.

SQLite session history loss is accepted (noted in v1.0 retrospective: "no session backup expected").

## Tasks

### 1. Replace SQLite with PostgresDb in `agentos/app.py`

```python
import os
from agno.storage.postgres import PostgresDb
from agno.memory.manager import MemoryManager

POSTGRES_DSN_SESSIONS = os.getenv("POSTGRES_DSN_SESSIONS")

# Replace: db = aiosqlite...
db = PostgresDb(db_url=POSTGRES_DSN_SESSIONS, create_schema=True)
```

### 2. Instantiate shared MemoryManager

```python
from agentos.model import chat_model

memory = MemoryManager(
    db=db,
    model=chat_model("cheap-worker"),  # memory summarization uses cheap model
)
```

### 3. Update agent factory calls to pass memory + knowledge

```python
from agentos.knowledge import make_knowledge, VaultKnowledge

kb = make_knowledge()
vault = VaultKnowledge(kb)
vault.load()  # one-shot startup load; incremental updates from curator

chat_agent = make_chat_agent(memory_manager=memory, knowledge=kb, db=db)
curator_agent = make_curator_agent(memory_manager=memory, db=db)
ingest_agent = make_ingest_agent(memory_manager=memory, db=db)
query_agent = make_query_agent(memory_manager=memory, knowledge=kb, db=db)
research_agent = make_research_agent(memory_manager=memory, knowledge=kb, db=db)
```

### 4. Wire AgentOS with MCP + A2A

```python
from agno.agent_os import AgentOS

agent_os = AgentOS(
    agents=[chat_agent, curator_agent, ingest_agent, query_agent, research_agent],
    teams=[supervisor_team],
    knowledge=[kb],
    enable_mcp_server=True,   # REQUIRED on constructor
    a2a_interface=True,
    tracing=True,
)

# CRITICAL: also pass enable_mcp_server=True to get_app()
app = agent_os.get_app(enable_mcp_server=True)
```

### 5. Update supervisor team in `agentos/agents/supervisor.py`

Replace inline `db` kwargs with shared `memory_manager`:
```python
def make_supervisor_team(memory_manager, db) -> Team:
    team = Team(
        ...,
        memory_manager=memory_manager,  # replaces inline MemoryManager instantiation
        output_schema=SupervisorRouting,
        ...
    )
```

### 6. Update tests

Update `tests/test_agentos.py` to mock `PostgresDb` and `MemoryManager`. Assert `enable_mcp_server=True` and `a2a_interface=True` on the `agent_os` instance.
