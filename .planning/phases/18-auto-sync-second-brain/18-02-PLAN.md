# Plan 18-02: Reindex helper + git-sync trigger + VPS deploy

## Goal

Create `scripts/reindex-vault.sh`, extend `scripts/git-sync.sh` to trigger it on `.md` changes,
and deploy both to the VPS. Closes the gap between `second-brain` git pulls and pgvector embeddings.

<task type="auto" id="18-02-A-reindex-script">
  <title>Create scripts/reindex-vault.sh</title>

  <files>
    - scripts/reindex-vault.sh (CREATE)
  </files>

  <action>
    Create `scripts/reindex-vault.sh` with the following content:

    ```bash
    #!/usr/bin/env bash
    # Flock-guarded pgvector reindex wrapper. Always exits 0 so cron is never broken.
    set -euo pipefail

    APP_DIR=/opt/ultra-agents-brain
    LOCK=/tmp/uab-reindex.lock
    LOG="$APP_DIR/logs/reindex.log"

    exec >> "$LOG" 2>&1
    echo "[$(date -u +%Y-%m-%dT%H:%M:%SZ)] reindex-vault: starting"

    # shellcheck source=/dev/null
    [ -f "$APP_DIR/.env" ] && . "$APP_DIR/.env"

    if flock -n "$LOCK" "$APP_DIR/.venv/bin/python" -m agentos.knowledge --reindex; then
      echo "[$(date -u +%Y-%m-%dT%H:%M:%SZ)] reindex-vault: complete"
    else
      echo "[$(date -u +%Y-%m-%dT%H:%M:%SZ)] reindex-vault: another reindex in progress; skipping"
    fi

    exit 0
    ```

    Make it executable: `chmod +x scripts/reindex-vault.sh`
  </action>

  <verify>
    File exists at scripts/reindex-vault.sh, is executable, contains `flock -n`, sources `.env`,
    and has `exit 0` as the final statement.
    Run: `bash -n scripts/reindex-vault.sh` (syntax check passes with no errors).
  </verify>

  <done>
    scripts/reindex-vault.sh created and syntax-validated.
  </done>
</task>

<task type="auto" id="18-02-B-git-sync-edit">
  <title>Extend scripts/git-sync.sh pull branch to trigger reindex on .md change</title>

  <files>
    - scripts/git-sync.sh (EDIT)
  </files>

  <action>
    In the `pull)` branch of `scripts/git-sync.sh`, insert HEAD capture before the merge and
    conditional reindex invocation after it. The edited pull block should read:

    ```bash
    pull)
      git fetch "$REMOTE" "$BRANCH" --quiet
      BEFORE=$(git rev-parse HEAD)
      git merge --ff-only "$REMOTE/$BRANCH" --quiet || {
        echo "git-sync: fast-forward failed; manual merge required" >&2
        exit 1
      }
      AFTER=$(git rev-parse HEAD)
      if [ "$BEFORE" != "$AFTER" ] && git diff --name-only "$BEFORE" "$AFTER" | grep -q '\.md$'; then
        "$(dirname "$0")/reindex-vault.sh" || true
      fi
      ;;
    ```

    The `push)` branch and all other branches remain unchanged.
  </action>

  <verify>
    Run: `bash -n scripts/git-sync.sh` (syntax check passes).
    Grep confirms `BEFORE=` and `AFTER=` are present in the pull branch.
    Grep confirms `reindex-vault.sh` is invoked inside an `if` guarded by `[ "$BEFORE" != "$AFTER" ]`.
  </verify>

  <done>
    scripts/git-sync.sh pull branch updated with conditional reindex trigger.
  </done>
</task>
