# Plan 11-03 — OBS-01 closeout: instrument `update_memory_task`

## Why

Phase 11 shipped with OBS-01 partially satisfied. `InstrumentedMemoryManager` wraps `create_user_memories` / `acreate_user_memories`, but the deployed chat agent runs with `enable_agentic_memory=True`, which routes writes through `MemoryManager.update_memory_task` / `aupdate_memory_task` (Agno `_default_tools.py:51`). That path bypasses our hook, so journal logs show no `OBS-01 memory write:` line even though `agno_memories` rows are written correctly. Confirmed twice: `11-01-SUMMARY.md` open items, re-confirmed in `11-02-SUMMARY.md` smoke evidence.

Closing this gap completes OBS-01 for the memory path — the only carryover from phase 11.

## Locked context (inherits from `11-CONTEXT.md`)

- D-08 — `InstrumentedMemoryManager(MemoryManager)` is the only OBS-01 hook point. We extend it; we do not introduce a second wrapper.
- D-08 log schema — `path=memory`, `agent_id`, `team_id`, `user_id`, `db_id`, `latency_ms`, `status`, `extracted_count`. On error: add `error_type`, `error_msg`.
- D-04 — file scope: `agentos/instrumented_memory.py` (edit) + `tests/unit/test_instrumented_memory.py` (new). No other files change.
- D-13 — purely additive; rollback = revert.

### Constraint discovered from Agno source

`MemoryManager.update_memory_task(task, user_id)` and `aupdate_memory_task(task, user_id)` take **only** `task` and `user_id` — no `agent_id` / `team_id`. The instrumented overrides therefore emit those fields as `None`. This is acceptable: the log line still contains `user_id` (the operationally meaningful tag for memory writes) and `extracted_count` (the row delta), which is what OBS-01 needs to prove the agentic-tool path executed.

If downstream consumers want `agent_id` on this path, that requires upstream changes to how Agno invokes the tool — out of scope. Tracked as a forward note in `Open items`.

## Tasks

### Task 1 — Override `update_memory_task` and `aupdate_memory_task` in `InstrumentedMemoryManager`

**File:** `agentos/instrumented_memory.py`

Add two methods next to `create_user_memories` / `acreate_user_memories`:

```python
def update_memory_task(self, task: str, user_id: Optional[str] = None) -> str:
    started = time.monotonic()
    db_id = getattr(self.db, "id", None) if self.db is not None else None
    status = "ok"
    error_type: Optional[str] = None
    error_msg: Optional[str] = None
    row_count_before = self._safe_count(user_id)
    try:
        return super().update_memory_task(task=task, user_id=user_id)
    except Exception as exc:
        status = "error"
        error_type = exc.__class__.__name__
        error_msg = str(exc)[:200]
        raise
    finally:
        row_count_after = self._safe_count(user_id)
        latency_ms = int((time.monotonic() - started) * 1000)
        self._emit(
            agent_id=None,
            team_id=None,
            user_id=user_id,
            db_id=db_id,
            latency_ms=latency_ms,
            status=status,
            error_type=error_type,
            error_msg=error_msg,
            extracted_count=max(0, row_count_after - row_count_before),
        )

async def aupdate_memory_task(self, task: str, user_id: Optional[str] = None) -> str:
    # same shape, awaits super().aupdate_memory_task(...)
```

**Constraints:**

- Reuse the existing `_safe_count` and `_emit` helpers — do not duplicate.
- Do NOT also override `run_memory_task` / `arun_memory_task`. Those are called *from inside* `update_memory_task`, so instrumenting both produces double log lines for one write. The outer task method is the right hook.
- Preserve method signatures exactly (positional + keyword compatibility with Agno's call sites).

### Task 2 — Unit test: override fires on synthetic call

**File:** `tests/unit/test_instrumented_memory.py` (new)

Write 4 tests. Use a fake DB stub and monkey-patch the parent's `update_memory_task` / `aupdate_memory_task` to bypass the LLM call.

```python
import logging
import pytest
from agentos.instrumented_memory import InstrumentedMemoryManager

class _FakeDb:
    def __init__(self, id_="ultra-brain-main"):
        self.id = id_
        self._rows = {}
    def get_user_memories(self, user_id=None):
        return list(self._rows.get(user_id, []))

def _mgr(db=None):
    m = InstrumentedMemoryManager.__new__(InstrumentedMemoryManager)
    m.db = db or _FakeDb()
    return m

def test_update_memory_task_emits_obs01_on_success(monkeypatch, caplog):
    mgr = _mgr()
    # Bypass real LLM call inside parent
    monkeypatch.setattr(
        "agno.memory.manager.MemoryManager.update_memory_task",
        lambda self, task, user_id=None: "ok",
    )
    with caplog.at_level(logging.INFO, logger="agentos.memory"):
        out = mgr.update_memory_task(task="remember teal", user_id="u1")
    assert out == "ok"
    msgs = [r.message for r in caplog.records if "OBS-01 memory write" in r.message]
    assert len(msgs) == 1
    assert '"path": "memory"' in msgs[0]
    assert '"user_id": "u1"' in msgs[0]
    assert '"db_id": "ultra-brain-main"' in msgs[0]
    assert '"status": "ok"' in msgs[0]

def test_update_memory_task_emits_obs01_on_error(monkeypatch, caplog):
    mgr = _mgr()
    def _boom(self, task, user_id=None):
        raise RuntimeError("boom")
    monkeypatch.setattr("agno.memory.manager.MemoryManager.update_memory_task", _boom)
    with caplog.at_level(logging.ERROR, logger="agentos.memory"):
        with pytest.raises(RuntimeError):
            mgr.update_memory_task(task="x", user_id="u1")
    msgs = [r.message for r in caplog.records if "OBS-01 memory write failed" in r.message]
    assert len(msgs) == 1
    assert '"status": "error"' in msgs[0]
    assert '"error_type": "RuntimeError"' in msgs[0]

@pytest.mark.asyncio
async def test_aupdate_memory_task_emits_obs01_on_success(monkeypatch, caplog):
    mgr = _mgr()
    async def _ok(self, task, user_id=None):
        return "ok"
    monkeypatch.setattr("agno.memory.manager.MemoryManager.aupdate_memory_task", _ok)
    with caplog.at_level(logging.INFO, logger="agentos.memory"):
        out = await mgr.aupdate_memory_task(task="t", user_id="u1")
    assert out == "ok"
    assert any("OBS-01 memory write" in r.message for r in caplog.records)

@pytest.mark.asyncio
async def test_aupdate_memory_task_emits_obs01_on_error(monkeypatch, caplog):
    mgr = _mgr()
    async def _boom(self, task, user_id=None):
        raise ValueError("nope")
    monkeypatch.setattr("agno.memory.manager.MemoryManager.aupdate_memory_task", _boom)
    with caplog.at_level(logging.ERROR, logger="agentos.memory"):
        with pytest.raises(ValueError):
            await mgr.aupdate_memory_task(task="t", user_id="u1")
    assert any("OBS-01 memory write failed" in r.message for r in caplog.records)
```

**Dependencies:** `pytest-asyncio` (check `pyproject.toml`; if missing, add it as a dev dep in the same commit).
