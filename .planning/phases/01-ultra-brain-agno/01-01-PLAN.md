# Phase 1 — Plan 1: Build ultra-brain on Agno

## Context

Replaces the hallucinated Hermes Docker image with a real, production-grade agent runtime.
The existing `ultra_brain/*.py` modules are WRAPPED (not rewritten) as Agno tools. A new
`agentos/` FastAPI package hosts all agents; `channels/telegram_adapter.py` becomes a dumb
adapter that long-polls Telegram and POSTs to AgentOS. Systemd services replace the old
`ultra-agents-brain.service`.

Architecture decision: Brain-first. `ultra-workshop` (coding agent team) is deferred until
this Brain is running and used daily for 2–4 weeks.

Source: `plans/continue-from-las-session-tender-wand.md` (imported 2026-05-19)

**Goal:** Both systemd services running on VPS.

**Files to create:**
