# Wave 2 Plan 5 — Research Agent + Knowledge Layer Reconfiguration

## Objective

Upgrade `agentos/agents/research.py` to orchestrator-tier with ReasoningTools, full memory stack, vault RAG, and `ResearchReport` output schema. Also rewrite `agentos/knowledge.py` to replace the stub `Knowledge(name=...)` with a real `PgVector`-backed knowledge base using local `SentenceTransformerEmbedder`.

Research is the highest-capability agent — it handles HITL-confirmed multi-step research tasks with reasoning.

## Context

The knowledge rewrite is bundled with research because research is the primary RAG consumer. After this plan, vault notes are semantically searchable for all agents (chat, query, research). The rewrite does NOT touch `agentos/tools/vault.py`.

## Tasks

### 1. Rewrite `agentos/knowledge.py`

Replace the current stub with:

```python
import os
from pathlib import Path
from agno.knowledge import Knowledge
from agno.vectordb.pgvector import PgVector, SearchType
from agno.embedder.sentence_transformer import SentenceTransformerEmbedder
from agno.reranker.sentence_transformer import SentenceTransformerReranker

VAULT_PATH = Path(os.getenv("VAULT_PATH", "vault"))
POSTGRES_DSN_KNOWLEDGE = os.getenv("POSTGRES_DSN_KNOWLEDGE")


def make_knowledge() -> Knowledge:
    vector_db = PgVector(
        table_name="vault",
        db_url=POSTGRES_DSN_KNOWLEDGE,
        embedder=SentenceTransformerEmbedder(
            id="sentence-transformers/all-MiniLM-L6-v2"
        ),
        search_type=SearchType.hybrid,
        reranker=SentenceTransformerReranker(),
    )
    return Knowledge(vector_db=vector_db)


class VaultKnowledge:
    def __init__(self, knowledge: Knowledge):
        self.knowledge = knowledge

    def load(self) -> None:
        for md_path in VAULT_PATH.rglob("*.md"):
            self.knowledge.insert(
                path=str(md_path),
                upsert=True,
                skip_if_exists=True,
            )

    async def aload(self) -> None:
        # Async variant for systemd-triggered refresh
        self.load()
```
