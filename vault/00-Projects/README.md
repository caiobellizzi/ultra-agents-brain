# Projects

Active efforts with a concrete outcome, review cadence, and eventual archive decision.

Each project should include:

- `_briefing.md`: current user-facing summary.
- `_log.md`: append-only project operations log.
- `_meta.yaml`: status, budget, owner, and review metadata.
- `sources/`: immutable source captures.
- `entities/`: project-local entity pages when a global page is not yet warranted.
- `concepts/`: project-local concept pages.
- `synthesis.md`: the current integrated view.
