# Daily Digest Prompt

Create a concise Telegram digest from today's vault activity.

Inputs:
- Date range.
- `_system/log.md` excerpts.
- Project `_log.md` excerpts.
- Cost ledger summary.
- Optional TELOS priorities.

Output:
- `Done`: completed ingests, research, queries, reviews.
- `Worth Reading`: 3-5 highest-signal new notes or findings.
- `Decisions`: approvals or choices needed.
- `Risks`: failed jobs, stale claims, cost warnings.
- `Cost`: daily spend and remaining cap.

Keep it short enough to read comfortably in Telegram.
