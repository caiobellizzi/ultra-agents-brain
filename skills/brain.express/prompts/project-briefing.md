# Project Briefing Prompt

Synthesize a finished research project into a Telegram-ready briefing.

Inputs:
- Worker summaries.
- `synthesis.md` draft or source excerpts.
- Project metadata and user request.
- Cost summary.

Output:
- One-line answer to the research request.
- 5-10 key findings with citations or source paths.
- Practical implications for the user.
- Open questions and low-confidence areas.
- Files written.
- Total cost and whether budget limits were hit.
